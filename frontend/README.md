
# Frontend - Irving's Mobile Order App

A responsive frontend application for Irving's Mobile Order App, built with HTML, CSS, and JavaScript.

## Project Structure

```
frontend/
├── index.html
├── assets/          # Images, icons, and other static resources
├── css/             # Stylesheets
└── js/              # JavaScript files
```

## Features

- Responsive mobile-first design
- Clean and intuitive user interface
- Integration with Spring Boot backend API

## Setup

1. Open `index.html` in your web browser
2. Ensure the backend API (Spring Boot) is running
3. Configure API endpoints in your JavaScript files

## Backend

This frontend communicates with a Java Spring Boot backend API. Ensure the API server is running before using the application.

## Browser Support

Works on all modern browsers (Chrome, Firefox, Safari, Edge)
